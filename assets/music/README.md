# Background music

Drop royalty-free / no-copyright `.mp3` (or `.wav`, `.m4a`) tracks in this folder.
The bot picks one at random for each Reel, loops/trims it to match the video
length, and fades it out at the end.

If this folder has no audio files, the bot still works — it just publishes a
silent video instead of failing.

Good sources for royalty-free tracks: YouTube Audio Library, Pixabay Music,
Free Music Archive (check each track's specific license before use).
